Android application for Execom workshop

This application represent TODO list in which you can add your tasks, and choose in which category you can add it, too. 
When you click plus button in right bottom, new window, in which you can set name, description and category of task, opens.
If you want to, you can change any task. You have to hold long on task you want to change or delete and menu appears. 
In that menu you can set task as done, too. When you set task done, checkbox assigned to task get checked.
In any point you can see all tasks, tasks in some category or finished tasks by sliding from left side of screen to middle.
All tasks are kept in database, so even if you end application, your data stay safe.

Hope you enjoy!
